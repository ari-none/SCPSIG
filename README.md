# SCPSIG

Simple Crappy Pseudo-Shader Image Generation.

A program in the form of a Python library to easily create funky pictures using shader-like code, inspired by the Steam game "Replicube".

## How to use
- 1: Download the package (or the repo entirely)
- 2: Create a Python script or use one of the pre-existing scripts (such as `main.py`)
- 3: Create a function you'll pass as an argument for the `SCPSIG.generate_image()`, which takes an `UV` object as its sole argument, and returns a `Color` object.
- 4: Run the script & look at your beautiful creation !

## Why does this exist ?
I saw a trailer for Replicube, and I liked how the models were made with code. So I decided to make something to create images using code similarly. Because it's fun.
